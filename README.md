# 麻豆 - MarkDo
MARK what you get, DO what you need

第三方Moodle客户端，使用Kotlin Multiplatform编写，支持Desktop和Android平台，计划未来支持iOS
## 使用了：
- [FetchMoodle](https://github.com/BiUH-C2G/FetchMoodle)：我社自研Moodle访问库，现已KMP化，欢迎采用
- Ktor、Ksoup：被FetchMoodle依赖，用作网络请求和网页解析
- Koin、Decompose、Room、DataStore：好用无需多言
## 展望
- 添加iOS支持
- 支持更多实用的Moodle站点功能
- 实现待办提醒和消息推送
